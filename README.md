# Painel Professor EE · PWA

Esta aplicação permite o registo de aulas, faltas e sumários por professores de educação especial.

## Funcionalidades
- Registo diário por professor
- Administração de professores, alunos, disciplinas e aulas
- Exportação de dados
- Sincronização com OneDrive via Microsoft Graph
- Instalação como app (PWA)

## Instalação no Telemóvel
1. Acede ao link da aplicação no browser (Chrome, Edge ou Safari)
2. Espera carregar a página principal
3. No Android: Menu → "Adicionar à tela inicial"
4. No iOS: Partilhar → "Adicionar à tela inicial"

## Publicação no GitHub Pages
1. Cria um repositório e ativa GitHub Pages
2. Faz upload dos seguintes ficheiros:
   - index.html
   - style.css
   - app.js
   - manifest.json
   - service-worker.js
   - icon-192.png
   - icon-512.png
   - README.md

## Requisitos
- Conta Microsoft para login
- Permissões de acesso ao OneDrive escolar
